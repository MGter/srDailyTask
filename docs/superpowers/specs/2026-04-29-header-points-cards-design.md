# 顶部积分小卡片设计

## 背景
当前顶部只显示用户总积分，用户希望在同一栏看到更完整的今日积分状态：当日累计、当日消耗、当日结余和总计。

## 设计
在 `web/src/views/Tasks.vue` 的顶部 `header` 中，保留标题、用户名和退出按钮，新增一组积分小卡片。

显示字段：
- 当日累计：今天收入积分，对应每日统计的 `earn`
- 当日消耗：今天支出积分，对应 `spend`
- 当日结余：今天净变化，对应 `balance`
- 总计：当前用户积分，对应 `user.points`

## 数据来源
复用现有 `pointsApi.getDailyStats(userId, { days: chartDays.value })` 返回的 `dailyStats`。
前端通过当前日期匹配今天的数据；没有数据时显示 0。
总计继续使用 `userApi.getUser` 返回的 `user.points`。

## UI
采用小卡片样式，放在顶部栏中间区域。
桌面端横向排列；手机端自动换行，保持紧凑。
颜色建议：累计绿色、消耗红色、结余橙色、总计蓝色。

## 修改范围
- `web/src/views/Tasks.vue`
  - template：替换当前单个积分显示为积分卡片组
  - script：新增 `todayStats` computed
  - style：新增顶部积分卡片样式和手机端适配

## 验证
- 刷新页面后顶部显示四个指标
- 添加收入/支出后，当日累计/消耗/结余和总计刷新
- 打卡后，当日累计和总计刷新
- 跳过任务不改变积分统计
- 手机端顶部不挤压，卡片可换行
