# 积分热力图设计

## 背景
用户希望在积分统计图下方增加类似 GitHub 贡献图的热力图，并参考截图中的图标排列方式。

## 设计
在 `web/src/views/Tasks.vue` 的积分统计卡片内，放在现有 Chart.js 图表下方。

热力图排列：
- 横向按周排列
- 纵向按星期排列
- 显示月份标签
- 单元格参考 GitHub 小方块风格

数据口径：
- 使用现有 `dailyStats` 数据
- 每天取 `balance` 作为颜色依据
- 正数越多越绿
- 负数越多越红
- 0 或无数据为浅灰

显示范围：
- 跟随现有天数选择器：7 / 14 / 30 / 180 / 360 天
- 不新增后端接口
- 不新增前端依赖

## UI
热力图放在积分图下方，作为同一个统计卡片的一部分。
桌面端横向显示；手机端允许横向滚动，避免小方块过窄。
悬停时通过 `title` 显示日期、累计、消耗、结余。

## 修改范围
- `web/src/views/Tasks.vue`
  - template：在 `chart-wrapper` 下方增加 `heatmap-section`
  - script：新增 `heatmapWeeks` computed 和颜色 class 方法
  - style：新增热力图网格、月份、星期、小方块样式

## 验证
- 切换 7/14/30/180/360 天，热力图范围同步变化
- 正结余显示绿色，负结余显示红色，0 显示灰色
- 鼠标悬停能看到日期和积分数据
- 手机端不挤压页面，可横向滚动
